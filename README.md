# Basic RayTracer
Built by following the [Ray Tracing in One weekend](https://github.com/a1Kamyk/raytracing-in-a-weekend.git) book, in order to understand raytracing and rendering better

Notable Features:
+ Material System: Lambertian (scatter), Metal, Dielectric (glass)
+ Hittable objects: Spheres and Planes
+ Flexible camera parameters
+ Multi-sampling

# Building requirements
+ C++ 20
+ CMake 3.12
+ GCC or MSVC (for OpenMP multithreading)
